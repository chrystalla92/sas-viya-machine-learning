Overview
--------

The MNIST Autoencoder package provides a comprehensive framework for:

* Loading and preprocessing MNIST digit data
* Defining and training autoencoder neural network architectures
* Evaluating model performance with various metrics and visualizations
* Saving and loading trained models

The package is built on PyTorch and follows modern Python packaging standards
with comprehensive testing and documentation.

Key Features
------------

* **Modular Design**: Organized into separate modules for data, models, training, and evaluation
* **Modern Architecture**: Uses PyTorch for flexible neural network implementation
* **Comprehensive Testing**: Full test coverage with pytest
* **Documentation**: Complete API documentation with examples
* **Type Safety**: Full type hints for better development experience
